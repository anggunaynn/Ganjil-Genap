# Ganjil-Genap
Membaca satu bilangan dari pengguna, kemudian menampilkan apakah bilangan tersebut ganjil atau genap.

Dengan kata lain, program ini membantu mengelompokkan bilangan berdasarkan sifatnya terhadap pembagian 2:

Bilangan genap → habis dibagi 2

Bilangan ganjil → tidak habis dibagi 2Tujuan program GanjilGenap.java adalah untuk melatih penggunaan input, operator modulus, dan percabangan if-else dalam menentukan apakah sebuah bilangan tergolong ganjil atau genap.
